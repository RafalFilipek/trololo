# Trololo

1. Clone
2. Create virtual env.
3. Activate virtual env.
4. `pip install -r requirements.txt`
5. Change `GITLAB_API_URL` in api.py ( _yep, no settings yet_ )
6. open `http://127.0.0.1:5000`
7. Get your private token from gitlat
